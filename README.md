## About this Project
This is an Inventory Management web application built with Express and PostgreSQL. The app allows users to manage categories and items for a fictional store of video game theme. Users can perform full CRUD operations on both categories and items, view items within each category, and navigate the inventory through a simple interface.

 ### Built with
 - Express
 - NodeJS
 - PostgreSQL
